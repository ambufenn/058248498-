Perencanaan & Persiapan Server: Landing Page Tugas 3

Dalam pengerjaan Tugas 3 ini, saya melakukan perencanaan server dengan pendekatan Serverless Architecture, di mana use case yang saya bangun adalah landing page akademik yang interaktif untuk menampilkan biodata dan jawaban kuis. Berikut adalah alur perencanaan dan persiapan yang saya lakukan:

Pemilihan Arsitektur Serverless: Saya memilih menggunakan Vercel karena arsitekturnya yang serverless.

Implementasi Workflow CI/CD: Workflow yang saya terapkan adalah CI/CD.

Struktur Arsitektur Sistem:

  • Front-End Layer: Disusun dengan Vercel Edge Network.
  • Back-End Layer: Menggunakan Serverless Function di folder /api.

Keamanan dan Manajemen: Keamanan terjamin dengan SSL/HTTPS dan GitHub sebagai version control.

Alur Data: Request -> Vercel -> File Statis/Serverless Function.

Kesimpulan: Berfokus pada otomatisasi dan arsitektur modern.
